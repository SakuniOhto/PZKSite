## Модальное окно для информации о преподавателе

### Что было реализовано:
1. **State управление** - добавлен `selectedTeacher` state для отслеживания открытого модального окна
2. **Интерактивность** - corner button теперь является кликабельной кнопкой, которая открывает модальное окно
3. **Модальное окно** - отображает полную информацию преподавателя из поля `fulldesc` с форматированием
4. **Анимации переходов**:
   - `fadeIn` (0.3s) - плавное появление затемнения фона
   - `slideUp` (0.4s) - окно появляется снизу с кубической кривой easing

### Структура модального окна:
- **Overlay** - полупрозрачный фон (rgba(0, 0, 0, 0.5))
- **Content** - основное окно с:
  - Кнопкой закрытия (✕) в верхнем правом углу
  - Заголовком с фото и именем преподавателя
  - Телом с полным описанием (fulldesc)

### Файлы изменены:
- `app/page.tsx` - добавлен state, кнопка, JSX для модального окна
- `app/globals.css` - добавлены стили для модального окна и анимации

### Особенности:
- Закрытие по клику на overlay или кнопку ✕
- Максимальная высота 80vh с прокруткой
- Адаптивная ширина (90% на мобильных, макс 800px)
- Использует существующую функцию `formatTeacherData()` для форматирования текста
