## Анимация пролистывания в блоках Преподаватели и Дисциплины

### Что было реализовано:
1. **Блок Преподаватели** - добавлена анимация горизонтального входа (slideInTeachers)
   - Карточки появляются с прозрачностью 0 и смещением на 30px вправо
   - Плавно переходят в видимое состояние за 0.6s
   - Используется cubic-bezier(0.22, 1, 0.36, 1) для естественного движения

2. **Блок Дисциплины** - добавлена анимация вертикального входа (slideDisciplines)
   - Карточки появляются с прозрачностью 0 и смещением на 20px вниз
   - Плавно переходят в видимое состояние за 0.6s
   - Тот же timing function для консистентности

### Технические детали:
- **CSS**: Добавлены @keyframes анимации в globals.css
- **React**: Добавлены key props для перезагрузки анимации при смене страницы
  - `key={`teachers-${currentPage}`}` для блока преподавателей
  - `key="disciplines-0"` и `key="disciplines-1"` для блока дисциплин

### Файлы изменены:
- `app/globals.css` - добавлены анимации
- `app/page.tsx` - добавлены key props для триггера анимации
